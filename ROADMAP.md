# Roadmap

CrunchyMurmur is developed in the open. This page communicates direction rather than a delivery promise; accepted work is tracked in GitHub issues.

## Current priorities

- Validate the first signed public release and automatic upgrade path on clean Windows, macOS, and Linux systems.
- Expand automated coverage for global shortcuts, recording overlays, permissions, and native audio behavior.
- Improve accessibility, keyboard navigation, editor ergonomics, and high-DPI layouts.
- Strengthen diagnostics without introducing telemetry or collecting user content.
- Evaluate portal-backed Linux system-audio capture across Wayland and X11 before enabling it.
- Prepare stable direct-download links for a future project website while keeping GitHub Releases as the file host.

## Distribution policy

Supported publication channels are GitHub Releases, direct links to immutable GitHub Release assets, and the verified terminal installers in this repository.

Microsoft Store and Apple App Store distribution are not currently planned. Packaging work should not weaken local-first behavior, privacy guarantees, or the reproducible GitHub release path.
