Whisper models

Models are stored in …. Downloads come straight from huggingface.co/ggerganov/whisper.cpp — no account needed. Pick the one that fits your accuracy / disk-space budget; for most people Large v3 Turbo (Q5) is the sweet spot.

Transcription

Choose where transcription runs. Local uses whisper-cli.exe on this machine — fully offline, no API key. Groq sends each clip to Groq's Whisper API — no model download, faster on most laptops, requires an account at console.groq.com.

Manage downloads in the Models tab.

Stored encrypted via Windows DPAPI (only this Windows user account can decrypt it). Get a free key at console.groq.com/keys.
